## Tools
- **SAST:** SonarQube, Semgrep, CodeQL.  
- **DAST:** OWASP ZAP, Burp Suite.  
- **SCA:** Dependency-Check, Trivy, Snyk.  
- **Secrets Management:** HashiCorp Vault, GitHub Secrets, AWS Secrets Manager.
